Whisper Transcription API

Official API for audio transcription based on OpenAI Whisper.

General Information

  • Base URL: https://transcription.wraiter.it
  • Supported audio formats: mp3, wav, ogg, flac, m4a
  • Supported languages: Multilingual with specific support for Italian
  • Technology: OpenAI Whisper, FastAPI

Note: This service is designed to provide accurate transcriptions of audio files using OpenAI's Whisper speech recognition model.

Endpoints

POST /transcribe/

Transcribes an audio file into text.

Request Parameters:

Parameter Type Required Description
audio_file File Yes Audio file to transcribe
language String No Language code (e.g. "it" for Italian). If not specified, it will be automatically detected

cURL Request Example:

curl -X POST "https://transcription.wraiter.it/transcribe/" \
  -H "Content-Type: multipart/form-data" \
  -F "audio_file=@file.mp3" \
  -F "language=it"

Python Request Example:

import requests

url = "https://transcription.wraiter.it/transcribe/"
files = {"audio_file": open("file.mp3", "rb")}
data = {"language": "it"}  # Optional

response = requests.post(url, files=files, data=data)
result = response.json()
print(result["transcription"])

Response Example:

{
  "transcription": "This is the transcribed text from the audio file."
}

Status Codes:

Code Description
200 Success. The response contains the transcription
400 Bad request. Missing file or unsupported format
413 File too large
500 Internal server error

Usage Limits

  • Maximum file size: 25MB
  • Maximum audio duration: 10 minutes
  • Rate limit: 60 requests/hour per IP address

Supported Models

The service uses OpenAI's Whisper model, with different variants available based on accuracy needs:

Model Description
tiny Lightest model, great for quick transcriptions with moderate accuracy
base Balances speed and accuracy (default)
small Higher accuracy at the cost of longer processing times
medium High accuracy for complex audio
large Maximum available accuracy

Integration

This API is designed to be easily integrated into any application that requires audio transcription functionality. The RESTful endpoint provides a standardized interface accessible from any programming language with HTTP support.

Privacy Policy

Uploaded audio files are processed exclusively to provide the transcription service and are not permanently stored on our servers. Files are automatically deleted after processing.
